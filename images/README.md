# Extension Icons

Place the following images in this directory:

## Required Images

### extension-icon.png
- **Size**: 128x128 pixels
- **Format**: PNG
- **Purpose**: Main extension icon shown in the Azure DevOps marketplace

### qualys-logo.png
- **Size**: 32x32 pixels
- **Format**: PNG
- **Purpose**: Task icon shown in pipeline editor

## Guidelines

1. Use the official Qualys logo/branding guidelines
2. Ensure the icon is clear and recognizable at small sizes
3. Use a transparent background if possible
4. Test the icon appears correctly in both light and dark themes
